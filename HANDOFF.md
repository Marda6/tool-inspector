# ENCY Inspector — отдельный проект

## Что это
Самостоятельный интерактивный прототип **панели инспектора** ENCY Core (CAM/ЧПУ, тёмная тема),
выделенный из главного окна (`C:\Users\ruslanm\ENCY 4`). Здесь — только инспектор: рейка вкладок
(Setup / Tool / Speeds / Strategy / Fixture / Links / More), заголовок и строки параметров,
плюс полный workflow вкладки **Tool** (сборка инструмента). Дерево операций, header, viewport,
фильтры и панель статусов сюда НЕ входят. Собрано на дизайн-токенах из Figma `Core`
(`fileKey = 5mNOE8ZjiF0VlYEhaeP4LF`).

## Файлы
- **`inspector.src.html`** — ИСТОЧНИК ПРАВДЫ, редактировать только его (ссылается на `assets/*.svg`).
- **`inspector.html`** — собранный самодостаточный файл (SVG вшиты). НЕ редактировать руками;
  пересобирать `node build.js` только для экспорта.
- **`assets/`** — 17 SVG-иконок (рейка + банк иконок строк), нужные инспектору.
- **`dev-server.js`** — live-сервер с авто-перезагрузкой (порт **5579**, отдельный от главного окна 5578).
- **`build.js`** — инлайнит ассеты в `inspector.html`.

## Как поднять живой просмотр
- `node dev-server.js`, открыть `http://localhost:5579`;
- или через preview-инструмент (`preview_start`).
Проверять изменения через `preview_eval` (DOM-состояние); скриншоты иногда таймаутят
из-за SSE-автоперезагрузки.

## Рабочие правила (важно)
- Дизайн-задачи ENCY/Core — через скилл **`ency-core-design`**. Не выдумывать элементы:
  собирать из компонентов и токенов Core; иконки тянуть из Figma, не рисовать.
- Правки — только в `inspector.src.html`. Никаких «магических» значений — только токены `--ec-*`.
- Заголовки секций — БЕЗ капслока и БЕЗ жирного (приглушать цветом `--ec-fg-56`).
- Текст UI — на английском.

## Состояние вкладки Tool (главное здесь)
Машина состояний `toolState`: `view` | `editAssembly` | `editTool` | `pickAssembly`.
- **view** — селектор сборки (`.tool-sel`, клик → режим `pickAssembly`) + read-only состав
  сборки (Station/Correctors/Orientation + группы Tool/Adapter с раскрытием) + блок
  «Used in operations».
- **pickAssembly** — полноэкранный список сборок с группами (Turret/Spindle), нумерацией T#,
  кнопкой Back, навигацией стрелками ↑/↓/Home/End и мгновенным применением (галочка едет за
  фокусом, Enter не нужен). Клик по строке = только выбор.
- **editTool** — режим подбора инструмента: параметры станции + фильтры (Tool/Holder/Insert/Hand
  Type) + поиск + список библиотеки, та же навигация стрелками.
Данные — в объекте `PARAMS.Tool` (`assembly[]` и `tool[]`), реальные ISO-названия пластин/державок.

## Связь с исходным проектом
Извлечено из `ency-main-window.src.html` (главное окно). Если правишь общие паттерны строки
(`buildRow`, токены, контролы) — имеет смысл синхронизировать оба проекта вручную.
Домен (Kit/Component/Adapter/сборка) — из tools-library.
